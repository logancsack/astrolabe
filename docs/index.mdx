Astrolabe is a headless OpenAI-compatible proxy for OpenClaw agents.

It classifies each request, routes it to the cheapest effective model policy target, and can escalate when confidence is low.

## What Astrolabe does

1. Classifies incoming chat requests with a low-cost model.
2. Routes trivial and moderate tasks to lower-cost models.
3. Escalates difficult requests to stronger models when needed.
4. Preserves OpenAI-compatible request and response format.
